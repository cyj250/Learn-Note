## 快速排序

**算法思想**

快速排序就是事先确定一个数作为轴，比这个轴小或等的放在左边，大的放在右边，每次执行这个过程可以将整个序列划分成两个部分，然后再对这两个部分采用相同的处理，即分治思想，最终返回有序的序列。算法的时间复杂度为O(n*logn)，是不稳定的，空间复杂度是O(logn)。

**代码实现**

```c
int quick(int *arr,int leftBound,int rightBound){
    int left = leftBound,right = rightBound - 1;
    int pivod = arr[rightBound];
    while(left<=right){
        while(left<=right&&arr[left]<=pivod) left++;
        while(left<=right&&arr[right]>pivod) right--;
        if(left<right) swap(arr,left,right);
    }
    swap(arr,left,rightBound);
    return left;
}

void sort(int *arr,int left,int right){
    if(left>=right) return;
    int mid = quick(arr,left,right);
    sort(arr,left,mid-1);
    sort(arr,mid+1,right);
}


void swap(int *arr,int a,int b){
    int tem = arr[a];
    arr[a] = arr[b];
    arr[b] = tem;
}
```
