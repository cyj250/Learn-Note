# 选择排序

**算法思想**

给定一个无序的数组，在未排序的序列中找出一个人最大或最小的元素，放在序列的开头，就是交换两个元素的位置，再在剩余的序列中进行这个过程。

**动画演示**

![img](https://www.runoob.com/wp-content/uploads/2019/03/selectionSort.gif)

**代码实现**

```c
void selectionSort(int *arr, int length){
    int i,j;//i记录最值元素的位置，j用于寻找
    int minPos;//存放找到的最值位置
    for(i=0;i<length-1;i++){
        minPos = i;//每次都认为第i个即为最值，然后再寻找改变。
        for(j=i+1;j<length;j++){
            if(arr[j]<arr[minPos]){//注意比较的是j和minPos，而不是i和j
                minPos = j;
        }
    }
    int tem = arr[i];
    arr[i] = arr[minPos];
    arr[minPos] = tem;
    }
}
```

**算法分析**

时间复杂度：O(n^2)

空间复杂度：O(1)

稳定性：不稳定
